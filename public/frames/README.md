# SR Chess Academy Cinematic Scroll Frames Directory

Please drag and drop your scroll frame images directly into this directory!

## Supported Naming and Formats
The cinematic scroll background is pre-configured to look for any of the following sequences with up to 100 frames:
- `frame_001.jpg` to `frame_100.jpg` (zero-padded 3 digits)
- `frame_01.jpg` to `frame_99.jpg` (zero-padded 2 digits)
- `frame_1.jpg` to `frame_100.jpg` (unpadded numbers)
- `001.jpg` to `100.jpg`
- `frame_001.png` to `frame_100.png`

Make sure to drag-and-drop or upload your files straight into this `/public/frames/` folder via the File Explorer toolbar in the editor interface!
